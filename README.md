# Toshiba T100 PC
This is a project aim to emulate Toshiba T100 PC and to unveil the structure of FAT 8-bit filesystem. Specifically Toshiba T100 TDISKBASIC
## Project Status
### Implemented
- Disassembler
- A Z80 CPU
- Event Logging System
